# dotfiles

```bash
git clone git@github.com:nshen/dotfiles.git ~/
~/dotfiles/install.sh
```


# hyper key

- `hyper + u`  chrome
- `hyper + i`  alacritty
- `hyper + hjkl` move
- `hyper + sdfe` raycast window management
- `hyper + []` next prev screen
